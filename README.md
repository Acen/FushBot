# FushBot
Trove fishing trainer made in CheatEngine with LUA interface.

## Requirements
This script has four requirements.

##### Must be in System32 Folder:
* libgcc_s_dw2-1.dll
* msvcrt.dll

##### Must be in CheatEngine root directory:
* lua5.1-32.dll
* winapi.dll
